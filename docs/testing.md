# Тестирование

[Архитектура](architecture.md) · [README](../README.md)

## Автоматические тесты

Запуск:

```powershell
npm.cmd test
```

Тесты покрывают:

- обновление и валидацию состояния;
- перемешивание букв внутри слов;
- сохранение первой и последней буквы в соответствующем режиме;
- расчет скорости вращения;
- поведение `Микс-трансформ`, включая выбор эффекта, направление вращения и отсутствие повторяющихся соседних состояний.

## Файлы тестов

| Файл | Что проверяет |
| --- | --- |
| `tests/state.test.js` | Начальное состояние, обновления и ограничения значений. |
| `tests/text-transformer.test.js` | Алгоритмы перемешивания слов. |
| `tests/animation.test.js` | Параметры анимаций и `Микс-трансформ`. |

## Ручная проверка

Перед завершением заметного изменения стоит проверить:

1. приложение запускается через `npm start`;
2. страница открывается в браузере;
3. каждый режим отображает текст без наложения букв и элементов управления;
4. `Пауза` останавливает анимации;
5. `Показать оригинал` временно возвращает нормальный текст;
6. демо-режим скрывает весь интерфейс кроме тренировочного текста и плашки `Esc - выйти`;
7. выход из демо-режима по `Esc` работает.
