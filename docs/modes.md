# Режимы тренировки

[Быстрый старт](getting-started.md) · [README](../README.md) · [Архитектура](architecture.md)

## Вращающиеся буквы

Каждая буква остается на своем месте, но вращается вокруг своей оси. Доступны два типа вращения:

- плавное;
- дискретное.

Скорость зависит от уровня сложности и дополнительного ручного множителя скорости.

## Микс-трансформ

Каждая буква остается на своем месте и детерминированно получает один из вариантов движения:

- вращение по часовой стрелке;
- вращение против часовой стрелки;
- бесконечную трансформацию между нормальным отображением, переворотом, зеркальным отражением и одновременным переворотом с отражением.

Последовательности трансформаций подобраны так, чтобы соседние состояния не повторялись и не создавали эффект зависания. Выбор эффекта стабилен для текущего текста и меняется при изменении seed-параметров состояния.

## Перемешанные слова

Все слова остаются на своих местах. Внутри каждого слова первая буква сохраняет исходную позицию, а остальные буквы перемешиваются.

Короткие слова, где перемешивание не дает заметного эффекта, остаются читаемыми без лишней обработки.

## Первая и последняя буква

Все слова остаются на своих местах. Внутри каждого слова первая и последняя буквы сохраняют исходные позиции, а средние буквы перемешиваются.

Этот режим ближе к классическому упражнению на чтение по форме слова.

## Демо-режим

Демо-режим показывает только тренировочный текст на весь экран. Панели ввода, настройки и кнопки скрываются.

В открытом демо-режиме остается только информационная плашка:

```text
Esc - выйти
```

Выход выполняется клавишей `Esc`.

## Управление

| Элемент | Назначение |
| --- | --- |
| `Пауза` | Останавливает анимации без сброса текста. |
| `Показать оригинал` | Временно показывает исходный текст без искажений. |
| `Демо-режим` | Переводит тренировку в полноэкранный режим страницы. |
| `Уровень` | Меняет базовую скорость движения. |
| `Скорость` | Позволяет точнее настроить темп анимации. |
