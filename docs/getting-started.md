# Быстрый старт

[README](../README.md) · [Режимы тренировки](modes.md)

## Требования

- Node.js;
- npm;
- современный браузер.

## Установка

```powershell
npm install
```

## Запуск

```powershell
npm start
```

По умолчанию приложение доступно по адресу:

```text
http://127.0.0.1:4173
```

Если порт занят, сервер выберет другой доступный порт и покажет его в консоли.

## Базовая проверка

```powershell
npm.cmd test
```

Ожидаемый результат: все тесты проходят без ошибок.

## Типовой сценарий

1. Введите короткий текст в левую область.
2. Выберите режим тренировки.
3. Настройте тип движения, уровень и скорость, если они применимы к выбранному режиму.
4. Используйте `Пауза`, чтобы остановить движение.
5. Используйте `Показать оригинал`, чтобы на короткое время увидеть нормальный текст.
6. Включите `Демо-режим`, если нужна тренировка на весь экран.
