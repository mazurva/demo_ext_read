# AGENTS.md

## Контекст проекта

Проект `demo_ext_read` - статический тренажер чтения и памяти на HTML, CSS и JavaScript.

Приложение позволяет ввести короткий текст и отобразить его в одном из тренировочных режимов:

- вращающиеся буквы;
- `Микс-трансформ`;
- перемешанные слова с сохранением первой буквы;
- перемешанные слова с сохранением первой и последней буквы.

Также есть пауза, временный показ оригинала и демонстрационный режим на всю страницу с выходом по `Esc`.

## Команды

```powershell

```

На Windows для тестов используйте `npm.cmd test`.

## Правила разработки

- Не логировать полный пользовательский текст.
- Не строить пользовательский текст через `innerHTML`.
- Для нового режима обновлять `MODE_VALUES`, `index.html`, `renderer.js`, тесты и документацию.
- Для визуальных режимов учитывать паузу и временный показ оригинала.
- Демо-режим не должен показывать кнопки или панели управления.
- Git в конфигурации AI Factory выключен, поэтому не полагаться на git-операции как обязательную часть рабочего процесса.
